🎯 Project Overview

Sentinels is a high-performance AI-powered disaster intelligence system designed to process real-time inputs (images, audio, and text) and deliver coordinated emergency response guidance. It simulates an intelligent control room operated by specialized AI agents working collaboratively to reduce disaster response time and save lives.

---

🚨 The Problem

During disasters:

* Emergency services are overwhelmed

* Information is fragmented

* Communication is chaotic

* Response delays cost lives

* Language barriers block critical help

---

✅ The Solution

A smart multi-agent ecosystem that:

* Detects disaster severity from images

* Interprets SOS voice calls

* Tracks weather & situational data

* Generates actionable evacuation plans

* Coordinates response strategies

---

🤖 The Five Sentinels

1. Vision Sentinel 🔍

Analyzes disaster images for:

* Structural damage

* Flood zones

* Fire spread

* Blocked roads

* Human presence

2. Audio Sentinel 🎤

Processes voice input for:

* Speech-to-text

* Language translation

* Emotion & urgency detection

3. Data Sentinel 📊

Monitors:

* Weather alerts

* Traffic conditions

* Emergency updates

* News feeds

4. Communication Sentinel 📱

Generates:

* Safety instructions

* Multilingual guidance

* Emergency alerts

5. Coordinator Sentinel 🧠

  The master controller:

* Aggregates all agent outputs

* Makes final emergency decisions

* Ensures response accuracy


---

🧠 Intelligent Response Pipeline

User Input (Image / Audio / Text)

→ Vision Sentinel (Gemini Vision)

→ Knowledge Base (RAG Emergency Data)

→ Coordinator Sentinel (Decision Engine)

→ Relief & Communication Agent

✅ Final Emergency Guidance to User

✅ Fast decision-making

✅ Context-aware response

✅ Fail-safe architecture

# Install Dependencies
